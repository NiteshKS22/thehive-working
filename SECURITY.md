# Security Policy (Internal LTS)

## Supported Versions
- TheHive 4.1.x (LTS)

## Reporting Issues
Report internal security findings to the Security Team via JIRA/Slack.
Do NOT report upstream as this version is EOL.

## Vulnerability Handling
- **Critical**: Fix within 24h or block deployment.
- **High**: Fix within 1 week.
- **Medium/Low**: Triage and fix in next monthly patch.
